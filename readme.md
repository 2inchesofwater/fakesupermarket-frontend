# Fake Supermarket

A small Eleventy- and Gulp-powered prototype for exploring “fake supermarket” storefronts and product data.

---

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm (comes with Node)

### Installation

From the project root:

```bash
npm install


## Quick start

From a terminal:

```bash
cd /path/to/this/project     # go to the project folder
npm install                  # first time on a new machine
npm start                    # start the frontend dev server


Then open:
http://localhost:8080

Note that the frontend runs on http:// while the backend requires https://
